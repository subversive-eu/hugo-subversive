## Css tricks

There is a lot of css options for class, use it like bootstrapp :

* options class="" :  
 * text-justify  
 * text-center  
 * no-deco ="list-style-type:none + text-decoration:none"  
 * float-left  
 * float-right  
 *  a ="cursor: pointer"  
 * Bootstrapp tricks :  
  * bg-dark  
  * bg-light  
  * text-light  
  * text-dark  
  * text-white  
  * text-warning  
  * text-success  
  * text-info
